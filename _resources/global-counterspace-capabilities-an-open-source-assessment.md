Drawing on publicly available information, this report provides an in-depth assessment of counterspace and anti-satellite capabilities being developed around the world. It assesses current and near-term capabilities for each country in the report across these categories: direct ascent, co-orbital, directed energy, electronic warfare, and cyber.
